# LEARNPLAYWRIGHTBATCH2X

A comprehensive learning repository for Playwright automation testing and JavaScript fundamentals. This repository contains practical examples, code snippets, and documentation to help developers master Playwright testing framework and JavaScript programming concepts.

## 📚 Repository Structure

### Chapter 01: Basic Concepts
- `01_Basic.js` - Fundamental JavaScript concepts and syntax
- `03_JSCommand.js` - JavaScript command line operations and process interactions

### Chapter 02: JavaScript Basics
- `01_identifires_rules.js` - Rules and examples for JavaScript identifiers (valid/invalid examples)
- `02_identifier.js` - Different naming conventions (camelCase, PascalCase, snake_case, etc.)
- `03_Shortcuts.js` - VS Code keyboard shortcuts organized by category
- `VS_Code_Keyboard_Shortcut_Windows.md` - Complete Windows shortcuts reference
- `VS_Code_Keyboard_Shortcut_Mac.md` - Complete Mac shortcuts reference

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/vinit199003/LEARNPLAYWRIGHTBATCH2X.git

# Navigate to the project directory
cd LEARNPLAYWRIGHTBATCH2X

# Install dependencies (if any)
npm install
```

### Running Examples
```bash
# Run JavaScript files
node chapter_01_basic/01_Basic.js
node chapter_02_JavaScriptBasics/01_identifires_rules.js

# View shortcuts
node chapter_02_JavaScriptBasics/03_Shortcuts.js
```

## 📖 Learning Topics Covered

### JavaScript Fundamentals
- **Identifiers**: Rules, naming conventions, valid/invalid examples
- **Variables**: Declaration with `var`, `let`, `const`
- **Data Types**: Primitives, objects, arrays
- **Operators**: Arithmetic, comparison, logical
- **Control Flow**: Conditionals, loops
- **Functions**: Declaration, expressions, arrow functions

### Playwright Testing
- **Basic Setup**: Installation and configuration
- **Browser Automation**: Launching browsers, navigation
- **Selectors**: CSS, XPath, Playwright-specific selectors
- **Actions**: Clicking, typing, waiting
- **Assertions**: Built-in and custom assertions
- **Page Objects**: Organizing test code
- **Test Organization**: Grouping and running tests

### Development Tools
- **VS Code Shortcuts**: Windows and Mac keyboard shortcuts
- **Debugging**: Breakpoints, stepping through code
- **Version Control**: Git commands and workflows
