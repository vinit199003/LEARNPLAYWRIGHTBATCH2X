# VS Code Keyboard Shortcuts - Mac

## File Operations

| Shortcut | Description |
|----------|-------------|
| Cmd + N | New File |
| Cmd + O | Open File |
| Cmd + K Cmd + O | Open Folder |
| Cmd + S | Save File |
| Cmd + Shift + S | Save As |
| Cmd + W | Close File |
| Cmd + Shift + W | Close Folder |
| Cmd + K W | Close All Editors |
| Ctrl + Tab | Switch to Next Tab |
| Ctrl + Shift + Tab | Switch to Previous Tab |
